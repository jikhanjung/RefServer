# 🗂️ RefServer 개발 로드맵

## 🎯 프로젝트 비전

RefServer는 **학술 연구 분야의 혁신적인 PDF 지능형 처리 플랫폼**에서 시작하여 **글로벌 연구 협업 플랫폼의 표준**이 되는 것을 목표로 합니다.

현재 상태: **v0.1.11 완료** - 하이브리드 언어 감지 OCR 시스템, ChromaDB 벡터 검색, 4층 중복 방지, 엔터프라이즈급 모니터링 완비

---

## 🚀 단기 개발 계획 (v0.1.12 - v0.2.0)

### **v0.1.12: 성능 최적화 및 백업 시스템 (예상 소요: 1-2주)**
**목표**: 대용량 데이터 처리 및 시스템 안정성 강화

- **⚡ ChromaDB 인덱스 최적화**:
  - HNSW 파라미터 자동 튜닝
  - 벡터 압축 및 메모리 최적화
  - 배치 검색 성능 개선 (10K+ 논문 대응)

- **🛡️ 백업 및 복구 시스템**:
  - ChromaDB 자동 백업 스케줄러
  - SQLite + ChromaDB 일관성 검증
  - 재해 복구 시나리오 구현

- **📈 성능 벤치마킹**:
  - 대용량 PDF 테스트 스위트
  - 동시 처리 부하 테스트
  - 메모리 누수 및 성능 병목 분석

- **📦 배치 업로드 시스템**:
  - 드래그앤드롭 다중 파일 업로드
  - ZIP/폴더 업로드 지원

- **🎯 스마트 큐 관리**:

  - 실패한 작업 자동 재시도 로직

- **📊 배치 분석 대시보드**:
  - 배치별 처리 통계 및 분석
  - 오류 패턴 분석 및 개선 제안

- **📊 검색 성능 최적화**:
  - 캐시 기반 빠른 검색 결과 제공
